# Design Lab

Рабочая папка для разработки и дизайна. Здесь применяются навыки с инфографики «42 навыка / 6 слоёв».

## Структура
- `src/` — исходный код (компоненты, страницы, приложения)
- `design/` — макеты, токены, дизайн-системы, спецификации
- `assets/` — картинки, иконки, шрифты, видео
- `docs/` — брифы, планы, гайдлайны бренда

## Карта: навыки с инфографики → реальные скиллы в этом чате

### Слой 1 — Frontend & UI (что уже есть)
| С инфографики | Реальный скилл |
|---|---|
| Frontend-design, Impeccable, Taste-skill | `frontend-design`, `impeccable`, `taste-skill` |
| Theme-factory, Design-motion, Animate | `ui-ux-pro-max`, `emil-design-eng` |
| Figma-implement, Figma-mcp | Figma MCP (`use_figma`, `get_design_context`) + `/figma-*` скиллы |
| Playwright-mcp / живой браузер | Browser-инструменты (preview, read_page, screenshot) |

### Слой 2 — Графика и видео
| С инфографики | Реальный скилл / инструмент |
|---|---|
| Canvas-design, Nano/Banana | генерация SVG/HTML-виджетов (`show_widget`), Figma assets |
| Remotion / Blender / AE | по запросу — код на React/Python/JSX |

### Слой 3–4 — Продукт и взаимодействие
| С инфографики | Реальный скилл |
|---|---|
| Hi-fi-mockups, Slide-decks | `present`, Figma Slides |
| Generative-ui, Progressive-reveal | `ui-ux-pro-max`, `frontend-design` |
| Brandkit-sync | `dataviz`, `impeccable` (токены) |

### Слой 5 — Промпты · Слой 6 — Безопасность и оценка
| С инфографики | Реальный скилл |
|---|---|
| System-structure, Constraint-spec | `brainstorming`, `writing-plans` |
| Guardrails, Quality-rubrics | `code-review`, `verification-before-completion` |
| Task-decomposition, Handoff | `subagent-driven-development`, `dispatching-parallel-agents` |

## Процесс работы (как я буду вести проекты здесь)
1. **Brainstorming** — сначала уточняем задачу и дизайн-направление
2. **Design** — токены, макет, направление (не «типовой AI-вид»)
3. **Build** — код по best practices, живая проверка в браузере
4. **Review** — code-review + проверка перед завершением
